# Mission

복수 워크스페이스에서 병렬 실행되는 AI 에이전트들을 하나의 앱에서 감독하고 결과물을 검증하는 데스크톱 앱.

## 해결하는 문제
- 에이전트가 뭘 하는지 실시간으로 파악하기 어렵다
- 에이전트 산출물(diff, 코드, 파일)을 빨리 검증할 도구가 없다
- 권한 요청이 분산되어 놓치거나 늦게 대응한다
- 여러 프로젝트의 에이전트를 각각 따로 관리해야 한다

## 목표
- 멀티 워크스페이스: 하나의 앱에서 여러 프로젝트를 관리
- 에이전트 상태·진행 상황 통합 감독
- 컨텍스트 바인딩된 검증 도구 (diff, 코드 뷰, 브라우저)
- 통합 승인 큐로 권한 요청 집중 처리

## 기술 방향
- 데스크톱 앱 (기술 스택 미정 — 열어놓고 결정)
- Claude Code CLI subprocess 통신 유지

## 타겟
여러 워크스페이스에서 복수 에이전트를 병렬 감독하는 시니어 개발자
