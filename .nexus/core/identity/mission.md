# Mission

AI 에이전트 팀을 설계·감독·운영하는 독자적 개발 도구.
시니어 개발자가 복수의 에이전트를 조율하고 결과물을 검증하는
**에이전트 오케스트레이션 워크스테이션**.

## 해결하는 문제
- 에이전트 팀의 의사결정 과정을 추적·시각화할 도구가 없음
- 승인·거부·수정이 필요한 에이전트 액션을 통합 관리하는 인터페이스 부재
- 에이전트 컨텍스트에 바인딩된 검증 도구(코드 뷰어, 브라우저, 파일탐색기) 필요

## 목표
- [meet]→[d]→[run] 미팅 기반 의사결정 워크플로우 1급 지원
- Mission Control: 통합 승인 큐 + 에이전트 상태 대시보드
- AgentBackend 추상화로 점진적 API 직접 호출 전환
- 검증 도구를 에이전트 컨텍스트에 바인딩하는 자유 레이아웃

## 타겟 사용자
복수 에이전트를 감독하는 시니어 개발자 (Agent Supervisor)

## 핵심 차별화
1. **미팅 기반 워크플로우** — 방법론이기 때문에 모방하기 어려운 차별점
2. **에이전트 감독 대시보드** — 통합 승인 큐 + 컨텍스트 바인딩 검증
