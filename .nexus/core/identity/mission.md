# Mission

Claude Code CLI의 강력한 기능을 GUI로 래핑하여,
에이전트 오케스트레이션 상태 시각화와 자유 레이아웃을
제공하는 독립 데스크톱 앱.

## 해결하는 문제
- 공식 Desktop App/VS Code 확장은 커스텀 패널 추가 불가
- Nexus 에이전트 상태(consult, decisions, tasks) 시각화 불가
- 마크다운 뷰어, 에이전트 타임라인 등 자유 레이아웃 필요

## 목표
- CLI stream-json 프로토콜 기반 완전한 GUI 래퍼
- 플러그인 기반 확장 가능 패널 시스템
- 파일 변경 안전성 (체크포인트 + 퍼미션 enforcement)
- Nexus 에이전트 오케스트레이션 1급 지원
