# Roadmap

## 접근 방식
기능 하나하나를 완성도 있게 쌓아나간다. MVP 스타일이 아닌, 각 기능이 온전하게 동작한 후 다음으로 넘어간다.

## 현재 우선순위

### 1. 아키텍처 설계
- 모듈 구조 설계 (책임 분리, 경계 정의)
- 통신 프로토콜 설계 (IPC, CLI 통신, 이벤트 흐름)
- 상태 관리 패턴 정의
- 에러 처리 전략

### 2. 기술 스택 결정 + 프로젝트 기반
- 프레임워크 리서치 및 선정
- 프로젝트 스켈레톤 셋업

### 3. 기능별 구현 (순서 미정)
각 기능을 완성도 있게 구현한 후 다음으로:
- CLI 통신 모듈
- 세션 관리
- 에이전트 추적
- 권한 처리
- 검증 도구
- 멀티 워크스페이스
- (순서는 아키텍처 설계 후 결정)
