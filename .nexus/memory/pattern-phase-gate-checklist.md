# pattern-phase-gate-checklist

스냅샷 날짜: 2026-04-24

Phase A 및 한국어 릴리스 게이트에서 도출한 재사용 가능한 체크리스트 패턴이다.

---

## 1. 카테고리

| 범주 | 코드 | 대상 |
|------|------|------|
| 네이티브 서명·신뢰 체인 | S | 서명, notarize, Dock·Finder 실행 |
| 셸·도구·환경 | E | PATH, brew, node, mise, LANG |
| 입력·표현·국제화 | K | IME 동작, 문자 폭, NFC 정규화, 입력 지연, 폰트 폰백 |
| 터미널·스크롤백·검색 | T | 다중 탭, FIFO, 복사·붙여넣기, 검색 경계 |
| 워크스페이스·프로세스·복원 | W | 워크스페이스 열기·닫기·전환, sidecar·node-pty 생명주기, 재시작 복원 |

---

## 2. 공통 항목

**S** — 서명 검증이 exit 0이다. 격리·notarize 경고 없이 실행된다. Dock 및 Finder 시작이 정상이다. arm64·x64를 모두 검증한다.

**E** — 앱 내부 터미널 PATH가 예상 접두사를 포함한다. brew·node·mise가 실제 경로로 해석된다. LANG이 UTF-8 로케일이다. 로그인 셸 환경이 의도대로 전달된다.

**K** — 대상 IME가 시스템 입력기로 선택된다. 조합 중 커서·오버레이가 캐럿과 정렬된다. 조합 중 Enter는 조합만 커밋하고 두 번째 Enter가 개행·실행을 유발한다. 한글이 2셀로 렌더링된다. NFD 입력 경로가 저장 시 NFC로 정규화된다. 수동 타이핑 반복 시 문자 누락이 없다. 입력→에코→페인트 평균 지연이 16ms 이하이다. 터미널 폰트 폰백 스택에 대상 언어 글꼴이 순서대로 포함된다.

**T** — 다중 탭 열기·닫기·전환이 정상이다. 스크롤백이 FIFO 방식이다. 선택 시 자동 복사 및 단축키 복사·붙여넣기가 cross-app에서 일치한다. 검색이 경계에 도달하면 no-more-matches를 표시하고 이전 탐색으로 복구 가능하다.

**W** — 워크스페이스 3개 이상을 열고 닫을 수 있다. 전환 시 터미널 cwd가 선택된 워크스페이스 절대 경로와 일치한다. 빠른 전환 시 렌더링이 안정적이다. sidecar·node-pty가 워크스페이스·탭 단위로 격리 생성된다. 앱 종료 후 남는 프로세스가 없다. 재시작 시 이전 워크스페이스와 탭 상태가 복원된다.

---

## 3. 증거 규칙

- 자동화 게이트: `bun test` 출력 또는 CI 로그가 유일 증거. 테스트는 파일을 쓰지 않는다.
- 수동 게이트: 아키텍처별로 별도 실행하고 스크린샷·동영상·텍스트를 아티팩트로 남긴다.
- 한 건의 FAIL은 전체 게이트를 BLOCK한다. N/A는 실행 전 메타데이터에서 명시 승인해야 한다.
- 체크리스트는 템플릿이다. PASS는 별도 verdict 문서에서 선언한다.
